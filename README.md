# Лабораторная 1 по дисциплине "Технологии программирования"

## Описание проекта
Проект для расчета среднего рейтинга студентов по дисциплинам.

## Функциональность
- Чтение данных из TXT и XML файлов
- Расчет среднего рейтинга студентов
- Поиск студентов с 100 баллами по всем предметам
- Модульное тестирование
- CI/CD с GitHub Actions

## Используемые технологии
- Python 3.10
- pytest для тестирования
- Git для контроля версий
- GitHub Actions для CI/CD

## Запуск проекта
```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск с TXT файлом
python src/main.py -p data/data.txt

# Запуск с XML файлом
python src/main.py -p data/data.xml -f xml
