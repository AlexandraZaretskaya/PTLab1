# Лабораторная работа №1: Система расчета рейтинга студентов

## Цель проекта
Разработка программы для обработки данных о студентах с расчетом среднего рейтинга и автоматическим тестированием.

## Основной функционал
Чтение данных из файлов формата TXT и XML  
Расчет среднего рейтинга студентов  
Поиск студентов с максимальными баллами по всем предметам
